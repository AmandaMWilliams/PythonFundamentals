#### Simple math 
* Automatically produces a float  

print(4)  // 4  
print(2 * 2)  // 4.0  
print(5.0 - 1)  // 4.0  

#### Exponents
print(2**5)  // 32.0  
print (9 ** (1/2))  // 3 (SquareRoot)  

#### Quotient
print(21 / 7) // 3 (division)  
print(20 // 7) // 2 (floor division)  

#### Modulo
print (20 % 7) // 6
